Herramienta de Desarrollo

Para este proyecto utilizamos Scratch, un entorno de programación visual desarrollado por el MIT Media Lab. Esta plataforma permite representar estructuras lógicas complejas mediante bloques gráficos, facilitando el aprendizaje de algoritmos sin la necesidad de escribir código complejo.

Componentes del Videojuego

están integrados por elementos fundamentales que permiten la interactividad:
Sprites y Fondos: Personajes y escenarios que cuentan con instrucciones específicas para definir su comportamiento y desplazamiento.
Sincronización:Uso de bloques de difusión (mensajes) para coordinar acciones paralelas entre múltiples objetos dentro del juego.
Interfaz de Usuario:Control del personaje principal mediante el teclado o el ratón.

Lógica y Funciones Programadas

Variables:Espacios de memoria utilizados para monitorear y actualizar en tiempo real el puntaje, las vidas y el tiempo restante.
Condicionales (if/else) y Bucles (for/while):Estructuras tipo "si... entonces" para la toma de decisiones automáticas (como ganar o perder) y ciclos "por siempre" para animaciones continuas
Detección de Colisiones:Programación de sensores que identifican cuando un personaje interactúa con obstáculos u objetos de valor, activando respuestas inmediatas.

Calidad y Pruebas

Cada videojuego pasa por un proceso de depuración para identificar y corregir errores en las rutinas programadas, asegurando que el producto final sea estable, funcional y fluido para el usuario.
